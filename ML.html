Project Description

This system is composed of four magnetic levitation systems. Each of them plays one of the Bass, Guitar, Drum and Vocal tracks respectively. User can interact with each of them to switch the track and play a special sound effect. The LED circle display the color corresponding with the music.

Implementation

Each of the ML system consists of a big cycle permanent magnetic for pushing the floated magnetic up, two perpendicular hall sensor for detecting the flex of the floated magnetic, four copper coils and the circuits for adjusting the posture of the floated magnetic to keep it at the center of the flex field and a circle of the LED for displaying.

Related Information

This project is for a US-China maker competition. We were a team of five.
I was working on hardware and others are working on software and appearance design.
The hardware was completely working and the interaction software was partially working.
This project won a $5000 prize and featured by several presses.
